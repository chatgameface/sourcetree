 iMachine님 오로카 github에 SkyRover 프로젝트 저장소가 있는데, 그곳에 파일을 올리고 관리하는 방법을 간단히 설명할려고 합니다. Git사용하기가 Command 라인으로 하는것은 아무래도 명령어를 입력하고 해야 해서 좀 복잡한 감이 있어서 제가 사용하고 있는 Git GUI 프로그램으로 진행하도록 하겠습니다.

 

 저는 아래 설명하는 프로그램과 Command라인을 혼합해서 쓰는데 처음 사용하기는 GUI방식이 편하고 접근하기도 쉬운것 같습니다.

 

 GitHub를 윈도우에서 사용하는것은 표윤석님 강좌에도 (http://cafe.naver.com/openrt/2320) 있으니 편한것을 사용하시면 될것 같습니다.

 

 

1. SourceTree 설치

 

 

 SourceTree는 무료로 사용할 수 있는 프로그램으로 윈도우(7이상)와 맥을 지원합니다. 라이센스는 Atlassian에 사용자 등록을 하면 무료로 사용가능합니다.

 

http://www.sourcetreeapp.com/

 

우선 위의 사이트에서 프로그램을 다운로드 하고 설치를 시작합니다.

 

 

 .NET이 필요함으로 중간에 설치 메뉴가 나오면 계속 진행을 하고 최종 설치가 완료됩니다.

 

 

2. 사용자 등록 

 

 SourceTree를 무료로 사용하기 위해서는 사용자 등록이 필요하며 아래 사이트에서 등록을 진행합니다.

 

https://id.atlassian.com/profile/signUp.action?application=mac&continue=https://my.atlassian.com/license/sourcetree

 

 

 

3. SourceTree 설정 

 

 프로그램을 처음 실행하게 되면 현재는 Git이 설치되지 않은 상태라서 Git을 어떤 방식으로 설치할지 선택을 해야 합니다. 여기서는 SourceTree 단독 사용을 위해 embedded 버젼의 Git 설치를 진행 합니다.  

 

 

선택을 하면 다운로드가 진행되는데 시간이 제법 걸려서 차한잔 하는 여유를 가져도 될것 같습니다.

 

 

Git설치 후에는 User정보 입력 및 라이센스 동의창이 나오고, Commit시 사용할 Full Name과 이메일 주소를 입력하고 Next를 누릅니다.

 

 

저장소에 특정 확장자의 파일은 무시하는 옵션을 사용할것이냐? 라는 묻는데 이때는 "No"를 해서 일단은 모든 파일을 저장소에 Commit할 수 있도록 합니다.

 

 

아래는 추측에는 GitHub와 데이터 송/수신시에 사용할 방식을 선택하는것 같은데 PuTTY/Plink를 선택합니다.

 

 

SSH Key를 가지고 있지 않음으로 "No"를 선택합니다.

 

 

이제 마지막으로 사용하고자 하는 Git 저장소에 로그인 할 수 있도록 username과 암호를 입력합니다. GitHub를 사용함으로 github 항목에 입력을 합니다.

 

 

드디어 SourceTree의 실행화면을 볼 수 있습니다.

 

 

 

4. SkyRover 프로젝트 저장소 추가

 

 이제 SourceTree를 사용하여 GitHub의 SkyRover 프로젝트 저장소를 로컬로 추가하도록 하겠습니다. 로컬에 추가후 로컬 파일을 수정하거나 추가하고 Commit과 Push의 과정을 거치면 GitHub 계정의 저장소에도 변경 내용이 반영되게 됩니다.

 

 

메뉴 아이콘 중에 Clone/New를 누르면 아래와 같은 팝업이 뜨고, 빨간색 원안의 지구본 아이콘을 클릭합니다.

 

 

 

기존 설정에서 GitHub 로그인 정보를 입력 하고, 윤석님이 oroca 저장소 권한을 드렸음으로 리스트 항목 중에 SkyRover_Projects란 항목이 보일겁니다. 해당 항목을 선택하고 OK를 누릅니다.

 

 

SkyRover_Projects를 선택하고, 아래 붉은 원안에 로컬에 복사할 폴더를 지정하고 Clone버튼을 누르면 GitHub 저장소에서 로컬 하드로 저장소를 복제합니다.

 

 

Clone을 완료하고 로컬 디렉토리에 가보면 GitHub의 SkyRover_Projects 파일들이 복제된것을 볼 수 있습니다.

 

 

5. 저장소에 파일 추가

 

 SkyRover 저장소에 관련 파일을 올리기 위해서는(캐드파일등) 우선 복제된 로컬 디렉토리에 원하는 디렉토리를 생성하고 추가하고 싶은 파일을 해당 디렉토리에 복사합니다.

 

 예제로 Doc 폴더 밑에 Exam폴더를 새로 만들고 그 안에 텍스트 문서 하나를 복사를 했습니다.

 

 

SourceTree를 실행하고 좌측의 File Status의 Working Copy를 선택하며, 아래 Working Copy Changes 항목에 위에서 추가했던 폴더와 파일의 변경 내역이 보입니다. 해당 변경 내역을 마우스로 Staged Changes 영역으로 이동을 시킵니다.

 

 

 

Working Copy의 변경 내용을 Staged 상태로 바꾼 후에 메뉴 아이콘에서 Commit을 누르고 Commit 메세지를 입력한 다음 Commit 버튼을 누릅니다.

 

 

 

Commit을 하더라도 실제 GitHub의 원격 저장소에 데이터가 업데이트 되는것은 아니고 그 중간단계에 있는 상태가 됩니다. 따라서 메뉴 아이콘 중에 Push를 해주어야 실제로 GitHub 원격지 저장소에 뎅이터가 업데이트 됩니다. Push아이콘을 눌러서 Push를 해줍니다.

 

 

그럼 https://github.com/oroca/SkyRover_Projects 에 가보면 폴더와 파일이 변경되어 있는것을 확인 할 수 있습니다.

 

 

 

6. GitHub 저장소 변경내용 적용하기

 GitHub의 저장소는 여러사람이 공동으로 작업을 할 수 있기때문에 다른 사람이 GitHub 저장소를 업데이트 했다면 로컬의 저장소에도 업데이트를 해주어야 합니다.

 

 Repository -> Refresh Remote Status 메뉴항목을 클릭하면

 

 

만약 GitHub 저장소에 변경내용이 있다면 Pull 아이콘 메뉴에 표시가 됩니다. Pull 아이콘을 누르면 로컬 저장소에 변경내용을 업데이트 하게 됩니다.

 

 

 

7. 마치며

 

  사진을 많이 붙이다 보니 좀 산만해 보이기는 한 것 같습니다. 혹시라도 잘못된 내용이 있으면 댓글로 알려주시면 수정토록 하겠습니다.

키워드가 포함된 새글이 등록되면 카페앱으로 알림을 받을 수 있습니다.사용방법

 

 

    본 게시물 작성자는 본인인증을 하지 않았습니다.

    상품등록게시판에 게시물을 작성할 때에는 온라인 아이디의 사용자가 본인임을 확인하는 '본인인증'을 거칩니다.
    단, 직접거래인 경우 본인인증을 나중에 할 수 있도록 제공하고 있습니다.

    본 게시글의 작성자는 개인정보보호를 위하여 일회용 안심 전화번호
    기능을 이용하고 있습니다.

    판매자 연락처 : 0
    연락 가능 기간 :
    • 일회용 안심 전화번호로 전화를 걸면 판매자의 실제 연락처로 연결되며,
      기존의 통화료 외 부가이용료가 없습니다.
      (판매자의 실제 연락처가 휴대폰 번호인 경우, 안심 전화번호 옆에
      휴대폰 아이콘이 표시되며 문자 발신도 가능합니다.)
    • 거래를 진행하실 경우 판매자의 실제 전화번호를 확인해두시는
      것이 좋습니다.
      (안전결제 이용 시에는 실제 전화번호 확인이 가능합니다.)

    에스크로 이체 방식이란?
    거래대금을 제3자 (에스크로 전문 업체)에게 맡긴 뒤
    물품 배송을 확인하고 판매자에게 지불하는 제도로써,
    물품을 받지 못했거나 반품할 경우에는 업체를 통해즉시
    환불해 주기 때문에 사기피해 등을 막을 수 있습니다.

    악성코드가 포함되어 있는 파일입니다.

     

    백신 프로그램으로 치료하신 후 다시 첨부하시거나, 치료가 어려우시면
    파일을 삭제하시기 바랍니다

    네이버 백신으로 치료하기

    고객님의 PC가 악성코드에 감염될 경우 시스템성능 저하,
    개인정보 유출등의 피해를 입을 수 있으니 주의하시기 바랍니다.

    저작권이 있는 파일입니다.

     

    본 파일은 저작권자의 요청에 의해 보호되고 있어 등록자 본인만 다운로드 가능합니다.

    저작권이 있는 파일의 등록 및 공유로 인해 저작권자의 권리침해 등의
    피해가 발생하지 않도록 주의하시기 바랍니다.

    고객님외에 다른 방문자에게는 이용이 제한되었습니다.

    이 파일은 저작권 침해의 우려가 있는 내용이 포함되어 있어
    고객님외의 다른 방문자가 다운로드 할 수 없도록
    이용이 제한되었습니다.

    이용제한 파일 :

    고객님외에 다른 방문자에게는 이용이 제한되었습니다.

    이 파일은 서비스 운영원칙에서 제한하는 내용이 포함되어 있어
    고객님외의 다른 방문자가 다운로드 할 수 없도록
    이용이 제한되었습니다.

    이용제한 파일 :

    고객님외에 다른 방문자에게는 이용이 제한되었습니다.

    이 파일은 저작권법 제103조(복제 전송의 중단)에 따라
    저작권자의 요청에 의해 고객님외의 다른 방문자가
    다운로드 할 수 없도록 이용이 제한되었습니다.

    이용제한 파일 :

    고객님외에 다른 방문자에게는 이용이 제한되었습니다.

    이 파일은 정보통신망이용촉진및정보보호등에 관한
    법률 제44조의2에 따라 관련 당사자의 요청에 의해
    사생활침해/명예훼손의 사유로 고객님외의 다른 방문자가
    다운로드 할 수 없도록 이용이 제한되었습니다.

    이용제한 파일 :

    구매자 안전거래 신청 안내

    전문업체의 에스크로 결제방식을 이용하여 판매자가 등록한 상품을 안심하고
    구매하실 수 있습니다.

    구매자 안전거래 이용 방법

    1. 1. 구매자 신청

      카페 게시물 내 구매자 안전거래 신청 버튼을 클릭하면
      유니크로 페이지에서 신청에 필요한 정보를 입력합니다.

    2. 2. 판매자 알림

      판매자에게 안전거래 신청을 받았다는 알림 메일과 문자
      메시지가 발송됩니다.

    3. 3. 판매자 등록

      판매자는 유니크로에 가입하여 메일에 기재된 인증번호로
      상품을 등록합니다.

    4. 4. 구매자 알림

      구매자에게 상품이 등록되었다는 알림 메일과 문자 메시지가 발송됩니다.

    5. 5. 구매자 결제

      구매자는 상품을 결제합니다.

    • 구매자 안전거래는 유니크로(㈜다우기술)의 책임하에 운영됩니다.
    • 구매자 안전거래 문의 : www.unicro.co.kr (1588-6295)
